# Arduino Uno Power Shield

A custom Arduino Uno Shield designed in Altium Designer to provide high-efficiency power to the Arduino Uno and additional high-power LEDs.

## Key Features
- 2-layer SMD PCB design in Altium Designer
- Fits exact dimensions of the Arduino Uno
- 12V DC input with ≥90% efficiency regulation to 5V
- Powers Arduino Uno and two 10W+ SMD LEDs (12V, parallel)
- Supports 16 AWG power input
- All components on top layer
- Standard 62 mil board thickness

## Design Focus
- Power electronics and voltage regulation
- High-current PCB routing and thermal management
- PCB layout optimization for compact shield design

## Project Files
- `1_Design_Files/` – Altium `.SchDoc` & `.PcbDoc`
- `2_Exports/` – Schematic, PCB layers, 3D view
- `3_Manufacturing/` – Gerber, Drill files, BOM
