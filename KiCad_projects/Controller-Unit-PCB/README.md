# Data Unit PCB – Flood Monitoring System

This PCB serves as the data transmission unit for a flood monitoring system, designed in KiCad as a reliable power and communication platform for ESP32-based sensors.

## Key Features
- Female connectors for ultrasonic HC-SR04 sensor integration  
- ESP32 microcontroller for wireless data transmission  
- Boost converter and 3.3V voltage regulation for reliable ESP32 operation  
- Integrated battery charging circuitry for portable deployment  
- 2-layer SMD PCB design in KiCad  

## Design Focus
- Reliable power delivery to ESP32 and sensors  
- Compact and efficient SMD layout  
- Integration of charging and boost conversion circuits for battery-powered operation  
- Modular design with sensor connectors for easy field installation
