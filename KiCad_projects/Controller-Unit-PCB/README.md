# Controller Unit PCB – Flood Monitoring System

This PCB serves as the data receiver and alert unit for a flood monitoring system, designed in KiCad to process data from remote sensor units and provide real-time alerts.

## Key Features
- Receives wireless data from Data Unit PCB via ESP32  
- 2-layer SMD PCB design in KiCad with integrated battery charging circuitry  
- Boost converter and 3.3V voltage regulation to reliably power ESP32-based systems  
- Real-time alerts using LEDs and buzzer for immediate notifications  
- ESP32 with USB-UART communication for monitoring and debugging  

## Design Focus
- Reliable power delivery and regulation for embedded sensors and alerts  
- Efficient SMD layout for compact, field-deployable design  
- Integration of battery charging, boost conversion, and alert circuits  
- Seamless wireless data reception and processing
